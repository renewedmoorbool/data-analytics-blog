Data Analytics Introduction [INCOMPLETE]

Data Analytics is the act of collecting a specific set of data in order to organize it and analyze it,
in order to draw proper conclusions related to a specific matter.

Some steps are required to achieve success in a data-driven project.
Data needs to be processed through a sequence:

  1. Data needs to be prepared and cleaned. This involves retrieving it, organizing it in a structure.
  2. Data has to be analyzed. This might involve statistical analysis.
    The analysis has to be centered around a specific topic,
    like a business question that may be asked.
